# Nyx Observability Platform

This solution is designed to be used as a baseline library for an observability platform. It provides a set of tools to be used in your development to enable standard and consistent features across your applications.

## Targeted Libraries

This solution is designed to have a few different observability related libraries that can be used in your applications. The intended libraries are as follows:

- Logging
- Monitoring

Each library is designed to be used independently, so you can pick and choose which libraries you want to utilize in your application.

This project is currently in the early stages of development, so expect to see more libraries added in the future.
